# Bursa Ariston - Professional Corporate Website 🛠️

This is a live commercial project developed for an Ariston authorized technical service provider in Bursa, Turkey. 

**Live Website:** [bursaariston.com](https://bursaariston.com/)

## 🚀 Project Overview
As the lead developer and designer of this project, I managed the entire lifecycle from conceptual design to live deployment. The goal was to create a trustworthy, SEO-friendly, and mobile-responsive platform for local customers.

## 🛠 Skills & Tools Applied
- **Platform:** WordPress (Customized for corporate identity)
- **Design:** UI/UX planning and layout structure
- **Graphic Design:** Created professional banners and visuals using **Canva**
- **SEO:** Optimized for local search results in Bursa
- **Maintenance:** Ongoing content management and security updates

## 📈 Key Results
- Fully responsive design compatible with all devices.
- High visibility in local search queries for technical services.
- Clean, professional user interface to build customer trust.
